Book cover Design Process:
This Cyber Security poster was created using Canva, combining my own layout, typography, and visual styling.
To enhance the overall look, I used a high-quality background image sourced from Google, and then customized it inside Canva with effects, overlays, and design elements to give it a modern, book-cover appearance.

Steps Followed:
1. Selected a futuristic cyber-security themed background image from Google.
2. Imported the image into Canva and applied layout adjustments, glow effects, and digital accents.
3. Designed the title and subtitle using bold, clean typography to match a book-cover style.
4. Added color enhancements, spacing adjustments, and composition refinement to give the poster a professional visual structure.
5. Exported the final design as a high-quality image suitable for book covers, presentations, and GitHub documentation.

  BOOKCOVER CYBERSECURITY

  How i designed this bookcover

  1.Collected the Background Image
    Searched on Google for a cyber-themed digital tech background, downloaded a high-resolution image, and imported it into Figma.

  2.Set Up the Figma Frame
    Created a book-cover–sized frame and placed the background image inside it, adjusting the crop and positioning to fit the layout.

  3.Added the Main Title
    Inserted text layers for the title, used bold clean fonts, applied white color for contrast, and center-aligned everything.

  4.Added the Subtitle
    Added the small descriptive subtitle in yellow to make it visible without overpowering the main heading.

  5.Added the Author Name
    Placed the author name at the bottom in all caps and centered it for balance.

  6.Final Adjustments & Export
    Tweaked spacing, alignment, and contrast for clarity, then exported the final design as a high-quality PNG
